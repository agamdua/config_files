Vim

My vim configuration relies on the fantastic Vundle to manage plugins.

  1. Clone vundle: $ git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim

  2. Now, run vim: $ vim (Replace vim with the alias if you use one).

Personally I open the .vimrc with vim instead of just launching vim ( $ vim ~/.vimrc )

  1. Quit vim with a :q and open whatever.

  2. Run a :BundleInstall

  3. Relaunch vim, and you're set to go.

Note: This is a cleanup of my previous .vimrc, taking only what I use. There are still a couple of packages I'm experiemeting with and a couple I have to pull in / fully configure.
